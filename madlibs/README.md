A fun word game where you fill in blanks with random words to create hilarious stories!

## Features

### GUI Version (madlibs_game.py)
- Beautiful graphical interface
- 5 different story templates
- Scrollable word input interface
- Copy story to clipboard
- Random story selection
- Play multiple rounds

### Console Version (madlibs_simple.py)
- Simple command-line interface
- Quick game mode for fast play
- Full game mode with story selection
- Colorful text support (optional)

## How to Play

1. **Choose a story** from the available templates
2. **Enter words** when prompted (nouns, verbs, adjectives, etc.)
3. **Read your hilarious story** with your words filled in!

## Installation

No installation required! Just run the Python files.

```bash
# For GUI version
python madlibs_game.py

# For console version
python madlibs_simple.py
```

## Requirements

- Python 3.6 or higher
- tkinter (comes pre-installed with Python)
- No external dependencies!
